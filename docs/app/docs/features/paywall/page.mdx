# Paywall

<Callout type="warning" title="Requires RevenueCat">
    The Paywall feature requires RevenueCat integration to be enabled during project setup.
    If you selected a preset without RevenueCat (like Minimal or Analytics-Focused), you will
    need to manually add the RevenueCat integration.
</Callout>

Monetization is built-in with RevenueCat integration. You can easily set up
subscriptions and one-time purchases.

## Displaying the Paywall

Use the `Paywall` component to show the subscription options to the user.

<CodeBlock
    code={`import { Paywall } from '@/components/Paywall';

export default function PremiumScreen() {
  return <Paywall />;
}`}
    language="typescript"
/>

<Callout type="info" title="RevenueCat">
    Make sure to configure your offerings and products in the RevenueCat dashboard.
