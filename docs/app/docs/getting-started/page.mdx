# Getting Started

Welcome to `create-core-stack`, the ultimate starter kit for building
production-ready mobile applications with React Native, Expo, and Node.js.

<Callout type="info" title="Prerequisites">
    Before you begin, ensure you have Node.js 18+ installed on your machine.
</Callout>

## Quick Start

The fastest way to get started is by using our CLI tool. It will guide you through
the setup process and generate a new project with your preferred configuration.

<CodeBlock code="npx create-core-stack@latest" language="bash" />

## What is included?

This stack comes pre-configured with everything you need to build a modern mobile app:

- **Expo Router** for file-based navigation
- **Zustand** for state management
- **Authentication** (Google, Apple, Email)
- **Backend API** with Fastify & Prisma
- **In-App Purchases** with RevenueCat
- **Analytics** with Adjust & Scate

## Next Steps

Once you have generated your project, check out the [Installation](/docs/installation) guide
to set up your development environment.
